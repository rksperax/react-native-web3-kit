# Contributing

Thanks for your interest in contributing to **react-native-web3-kit**!

## Getting started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/react-native-web3-kit.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feat/my-feature`

## Development workflow

```bash
npm test        # run unit tests
npm run lint    # type-check
npm run build   # compile to dist/
```

## Pull request guidelines

- Keep PRs focused — one feature or fix per PR
- Add tests for new utilities
- Update README if you add public API
- Follow existing TypeScript style (strict mode, no `any`)

## Code of conduct

Be respectful and constructive. This project follows the standard open-source community guidelines.
